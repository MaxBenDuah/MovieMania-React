## Run Locally

Clone the project

```bash
  git clone https://github.com/MaxBenDuah/MovieMania-React.git
```

Navigate to the project directory

```bash
  cd MovieMania-React
```

Run the project in your browser

```bash
  npm run dev
```

Open the project in your browser
